# Piedra_Papel_Tijeras
Proyecto: juego de piedra, papel o tijeras
Este es un juego de Piedra, Papel, Tijera, Lagartija, Spock; para la victoria, tienes que ganar 5 rounds!

#Selecciona 
              '0', para 'piedra, 
              '1', para 'papel', 
              '2', para 'tijera', 
              '3', para 'lagartija', o
              '4', para 'Spock'
              
               ¡¡Buena suerte!!
